## 摘要
非对比增强胸部 CT 为常规肺部筛查和机会性肺外筛查提供了丰富的机会。虽然多任务学习 (MTL) 可以统一这些不同的任务，但标准的硬参数共享方法在对不同病理进行建模时往往效果欠佳。我们提出了 HyperCT，这是一个通过超网络 (Hypernetwork) 动态调整 Vision Transformer 骨干网络的框架。为了确保计算效率，我们集成了低秩自适应 (LoRA)，使模型能够回归特定任务的低秩权重更新，而非全量参数。在涵盖放射学和心脏病学任务的大规模数据集上，HyperCT 的表现优于多种强基准模型，为全面的患者评估提供了一种统一且参数高效的解决方案。我们的代码可在 https://github.com/lfb-1/HyperCT 获取。

## Abstract
Non-contrast chest CTs offer a rich opportunity for both conventional pulmonary and opportunistic extra-pulmonary screening. While Multi-Task Learning (MTL) can unify these diverse tasks, standard hard-parameter sharing approaches are often suboptimal for modeling distinct pathologies. We propose HyperCT, a framework that dynamically adapts a Vision Transformer backbone via a Hypernetwork. To ensure computational efficiency, we integrate Low-Rank Adaptation (LoRA), allowing the model to regress task-specific low-rank weight updates rather than full parameters. Validated on a large-scale dataset of radiological and cardiological tasks, \method{} outperforms various strong baselines, offering a unified, parameter-efficient solution for holistic patient assessment. Our code is available at https://github.com/lfb-1/HyperCT.

### 2. 论文提出的方法论：HyperCT
*   **核心思想**：利用**超网络（Hypernetwork）**根据任务标识（Task Identity）动态生成 **Vision Transformer (ViT)** 骨干网络的权重。
*   **关键技术细节**：
    *   **低秩自适应集成（LoRA + Hypernet）**：为了解决超网络输出维度过大（ViT 参数量巨大）的问题，HyperCT 不直接生成全量权重，而是生成 LoRA 的低秩分解矩阵 $A$ 和 $B$。这使得超网络的参数复杂度从 $O(D^2)$ 降低到 $O(D)$（$D$ 为隐藏层维度）。
    *   **任务与位置嵌入**：输入包括可学习的任务嵌入（Task Embedding）和模块位置指示器（Module Indicator），后者告诉超网络在 ViT 的哪个层应用生成的权重。
    *   **动态权重生成**：对于 ViT 中的每个目标模块（如 Attention 和 MLP 层），超网络输出特定的低秩更新 $\Delta W = BA$，并将其叠加在预训练且冻结的骨干网络权重上。
*   **算法流程**：输入 CT 图像 -> 选择任务 ID -> 超网络生成该任务的 LoRA 权重 -> 注入冻结的 ViT 骨干 -> 输出预测结果。

*   **优化器**：AdamW，初始学习率为 1e-5，在第 15 个 epoch 时衰减。

*   **实验组数**：

    *   在前瞻性队列上进行了 7 项心脏任务的跨机构验证。
    *   **消融实验**：包括 LoRA 模块选择（仅 Attention、仅 MLP 或全部）、LoRA 秩（Rank）的大小影响、以及不同骨干网络（2D DINOv3 vs. 3D CTViT）的对比。
*   **充分性评价**：实验设计非常充分且客观。通过跨机构（外部验证）和跨时间（前瞻性验证）的测试，证明了模型的鲁棒性。同时使用了决策曲线分析（DCA）来评估临床实用性，而不仅仅是 AUC 指标。

### 6. 主要结论与发现
*   **性能优越**：HyperCT 在回顾性和前瞻性评估中均优于所有 MTL 基准，且在泛化性能上超过了资源消耗更大的 STL 模型。
*   **任务协同**：PCA 可视化显示，超网络自动学习到了临床相关的任务聚类（如心脏功能任务聚在一起），证明了模型能捕捉解剖学和生理学上的内在联系。

### 7. 优点与亮点
*   **参数高效**：通过 LoRA 结合超网络，在不增加推理时参数负担的情况下实现了极高的任务灵活性。
*   **统一框架**：首次将 18 项常规放射学任务与 7 项机会性心脏任务统一在一个模型中。

*   **解剖域限制**：目前模型主要针对心肺区域。如果扩展到完全无关的领域（如腹部的骨质疏松或肌肉减少症），可能需要重新训练或调整超网络结构。

*   **切片处理**：模型采用 2D 切片堆叠的方式处理 3D 卷，虽然利用了强大的 2D 预训练模型，但可能损失了部分精细的 3D 空间连续性信息。
