## 摘要
我们提出了特征对齐 CNN (FA-CNN)，这是一种通过端到端特征对齐实现内在类别归因的原型 CNN 架构。我们的直觉是，使用诸如线性层 (Linear) 和二维卷积层 (Conv2D) 等无序操作会导致语义概念不必要的重排和混合，从而使原始特征图难以理解。我们引入了两种新的保序层：阻尼跳跃连接 (dampened skip connection) 和全局平均池化分类器头 (global average pooling classifier head)。这些层强制模型保持从原始输入像素一直到最终类别 Logit 的端到端特征对齐。这种端到端对齐通过允许原始特征图内在展示类别归因，增强了模型的可解释性。我们从理论上证明了 FA-CNN 的倒数第二层特征图与 Grad-CAM 显著图是等价的。此外，我们证明了这些特征图随着网络深度的增加逐层缓慢演变，展示了特征在网络深度中向倒数第二层类别激活演进的过程。FA-CNN 在基准图像分类数据集上表现良好。此外，在像素移除百分比的可解释性任务中，我们将平均后的 FA-CNN 原始特征图与 Grad-CAM 和排列方法进行了对比。最后，我们对本工作进行了讨论并展望了未来，包括局限性以及向混合模型的扩展。

## Abstract
We present Feature-Align CNN (FA-CNN), a prototype CNN architecture with intrinsic class attribution through end-to-end feature alignment. Our intuition is that the use of unordered operations such as Linear and Conv2D layers cause unnecessary shuffling and mixing of semantic concepts, thereby making raw feature maps difficult to understand. We introduce two new order preserving layers, the dampened skip connection, and the global average pooling classifier head. These layers force the model to maintain an end-to-end feature alignment from the raw input pixels all the way to final class logits. This end-to-end alignment enhances the interpretability of the model by allowing the raw feature maps to intrinsically exhibit class attribution. We prove theoretically that FA-CNN penultimate feature maps are identical to Grad-CAM saliency maps. Moreover, we prove that these feature maps slowly morph layer-by-layer over network depth, showing the evolution of features through network depth toward penultimate class activations. FA-CNN performs well on benchmark image classification datasets. Moreover, we compare the averaged FA-CNN raw feature maps against Grad-CAM and permutation methods in a percent pixels removed interpretability task. We conclude this work with a discussion and future, including limitations and extensions toward hybrid models.

### 论文总结：End-to-end Feature Alignment (FA-CNN)

#### 1. 核心问题与整体含义（研究动机和背景）
传统的卷积神经网络（CNN）虽然性能强大，但其内部特征表示具有“纠缠性”（Entanglement），即单个神经元往往响应多个不相关的语义概念，且这些概念在层与层之间会被线性层和卷积层不断打乱和混合。这导致原始特征图对人类而言难以理解，必须依赖 Grad-CAM 等后验（Post-hoc）解释方法。
本论文的核心动机是设计一种**具有内在可解释性**的架构。通过强制实现“端到端特征对齐”，使网络从输入到输出始终保持语义顺序，从而让原始特征图直接具备类别归因能力，无需额外计算即可展示模型“看到了什么”。

#### 2. 方法论：核心思想与关键技术
FA-CNN 的核心思想是**保序性（Order-preservation）**，通过两个关键组件确保特征在深度传播中不被洗牌：

*   **阻尼跳跃连接（Dampened Skip Connection）：**
    传统的残差连接无法保证特征变化的幅度。作者引入了阻尼系数 $\beta = 1/L$（$L$ 为当前层深），公式为：
    $X_L = ReLU((1-\beta)X_{L-1} + \beta \tanh(F(X_{L-1})))$
    这种设计确保了每一层对整体特征幅度的贡献是均等的，并从理论上证明了**有界增量属性（Bounded Increment Property）**，使得特征图随深度增加而平滑演变（Morphing）。
*   **3D 全局平均池化分类头（3D Global Average Pooling Classifier Head）：**
    取消了会打乱特征顺序的全连接层。将倒数第二层的特征图按类别进行分组（每组 $R$ 个通道），直接通过空间和通道维度的平均池化得到类别 Logit。
*   **理论证明：**
    *   **Grad-CAM 等价性：** 论文证明了 FA-CNN 倒数第二层的原始特征图在数学上等同于该类别的 Grad-CAM 显著图。
    *   **逐层演化：** 证明了特征图在网络深处是逐渐从输入图像向最终的类别归因图“变形”的，提供了模型决策过程的动态视角。

#### 6. 主要结论与发现

*   **内在归因：** 证明了无需后处理即可获得高质量显著图，且这些图在像素移除任务中表现出与 Grad-CAM 相当的忠实度。
*   **深度演化：** 观察到模型在浅层会关注背景等干扰因素，随着深度增加，特征图会平滑地聚焦到目标物体上，揭示了模型消除噪声的过程。

#### 7. 优点
*   **理论完备：** 提供了严谨的数学证明，将内在架构设计与流行的后验解释工具（Grad-CAM）统一起来。
*   **端到端透明：** 实现了特征的逐层对齐，使得“黑盒”模型内部的特征演变变得肉眼可见。
*   **计算高效：** 在推理时直接输出解释图，无需像 RISE 或 K-Occlusion 那样进行数千次前向传播。

*   **负向归因缺失：** 由于使用了 ReLU 激活函数，模型目前只能识别正向贡献（红色区域），无法识别抑制类别的负向特征（蓝色区域）。

*   **规模限制：** 实验主要集中在 ImageNet-100，尚未在完整的 ImageNet-1K 或更大规模的视觉 Transformer（ViT）架构上进行大规模验证。
