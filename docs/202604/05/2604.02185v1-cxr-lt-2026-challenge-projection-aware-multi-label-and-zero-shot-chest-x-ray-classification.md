## 摘要
本挑战赛旨在解决已知胸部 X 光（CXR）病变的多标签分类以及未知病变的零样本分类问题。为了处理多样化的 CXR 投影方式，我们通过分类网络将特定投影模型集成到一个统一的框架中。针对零样本分类（任务 2），我们扩展了 CheXzero，提出了一种结合对比学习、非对称损失（ASL）和 LLM 生成的描述性提示词的新型双支路架构。这有效地缓解了严重的长尾不平衡问题，并最大化了零样本泛化能力。此外，强大的数据增强和测试时增强（TTA）确保了两个任务的鲁棒性。

## Abstract
This challenge tackles multi-label classification for known chest X-ray (CXR) lesions and zero-shot classification for unseen ones. To handle diverse CXR projections, we integrate projection-specific models via a classification network into a unified framework. For zero-shot classification (Task 2), we extend CheXzero with a novel dual-branch architecture that combines contrastive learning, Asymmetric Loss (ASL), and LLM-generated descriptive prompts. This effectively mitigates severe long-tail imbalances and maximizes zero-shot generalization. Additionally, strong data and test-time augmentations (TTA) ensure robustness across both tasks.

### 2. 方法论
研究团队提出了一个统一的框架，针对两个任务采用了不同的策略：

#### **任务 1：投影感知多标签分类**
*   **路由机制：** 使用 EfficientNet-B0 作为路由器，自动识别图像投影方向（AP/PA 或 Lateral），并将其分发至对应的专用模型分支。
*   **模型集成：** 每个分支集成三种架构：
    *   **PCAM：** 基于 ConvNeXt-v2，利用概率类激活映射（PCAM）池化生成标签热图。
    *   **CaiT：** 在冻结的 ConvNeXt 特征上应用 Transformer 建模。
    *   **Swin Transformer：** 利用分层自注意力处理高分辨率输入。

#### **任务 2：双支路零样本架构**
*   **核心思想：** 扩展 CheXzero 框架，结合对比学习与非对称损失（ASL）。
*   **关键技术：**
    *   **双支路损失：** 总损失 $L_{total} = L_{con} + \alpha L_{ASL}$。ASL 通过调整聚焦参数（$\gamma_- > \gamma_+$）来抑制易分类的负样本，增强对稀有正样本的敏感度。
    *   **LLM 描述性提示词：** 使用大语言模型生成详细的病理描述（如将“骨质减少”扩展为“弥漫性骨密度降低”），而非仅使用类名。
    *   **动态洗牌：** 训练时随机打乱多标签描述的顺序，防止模型产生词汇过拟合。

### 6. 主要结论与发现
*   **投影感知的优势：** 专门针对侧位（Lateral）投影训练模型能有效保留视角特定特征，比混合训练更具鲁棒性。

*   **实用性强：** 考虑了临床中常见的投视图差异，路由机制简单高效。
*   **零样本创新：** 结合 ASL 解决对比学习在长尾数据上的不稳定性，是一个极具启发性的改进。
*   **推理高效：** 尽管使用了集成和 TTA，推理延迟仍保持在毫秒级，符合临床实时诊断需求。

### 8. 不足与局限
*   **侧位片性能瓶颈：** 实验显示侧位投影（L）的性能远低于正位（AP/PA），尽管采用了专用模型，提升空间依然有限。
